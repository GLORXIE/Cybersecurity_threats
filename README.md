# 🛡️ Global Cybersecurity Threat Analysis (2015-2024)

### Overview
This project is a comprehensive Exploratory Data Analysis (EDA) of 3,000 global cybersecurity incidents. The goal was to identify patterns in attack vectors and their financial/operational impacts over a 10-year period.

### Key Features
* **Data Cleaning:** Handled datatypes and verified zero missing/duplicate records.
* **Preprocessing:** Implemented Label Encoding for categorical variables and Datetime conversion for time-series analysis.
* **Visualizations:** Univariate, Bivariate, and Correlation Heatmaps using Seaborn and Matplotlib.
* **Trends:** Analysis of financial loss and attack frequency from 2015 to 2024.
